A few things I've built

Lassen's Fly Shop

The purpose of this simple eCommerce-style application was to deepen my understanding and skillset in two areas: Use of the Next.js framework, and use of specific AWS tools (API, Lambdas, and DynamoDB) for data management on the back end. State is managed via the useContext hook. Styled using Material UI.

Coding Glossary App

Commissioned by the VP of Education at Code Fellows, this app allows enrolled students to maintain a glossary of terms they have learned. Logged in users can add, update, and delete terms. A 3rd party API is used for spell-checking. App built with React, Node.js, Express, and MongoDB.

  • See the Code

TV Show Review App

This app allows a user to log in (or create an account), view all TV shows existing in the database, view recommendations for shows provided by other users, add a new show to the database by searching a 3rd party API, add a review, and maintain a personal library of watched shows. Built with React, Redux, PostgreSQL, and Express. Styled using Material UI.

What I'm Working On

In my previous career as a project manager for retail marketing, I was often tasked with managing nationwide installations of displays and signage. I quickly identified the need for a central portal, where all vital information surrounding such a rollout could be easily tracked and reported to the client.

Therefore, I'm building an app that will address this need. The frontend will be built with React and Next.js, while the backend will consist of an Express server performing CRUD operations on a MongoDB database.
